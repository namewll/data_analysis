# data_analysis
Python 数据分析实战项目集合

## 技术栈
Python 3.x、Pandas、NumPy、Matplotlib、Pyecharts、Tableau、NLP 自然语言处理、Excel 自动化、数据可视化

## 项目内容
自动化办公：Python 批量生成薪酬报表，替代人工 Excel 操作
商业数据分析：电商经营数据清洗、奶茶店营收趋势分析、Python 岗位招聘数据可视化
文本挖掘：大众点评餐饮评论爬取与分析、飞猪酒店评论情感极性分析、NLP 分词与词云图绘制

## 亮点
覆盖自动化办公、商业分析、文本挖掘三大核心场景，从基础数据处理到进阶 NLP 应用形成完整链路，代码模块化可复用，适合作为数据分析与工程实践方向的学习成果展示。
